Turbine

Turbine is an easy to set-up and manage load-balancer and reverse proxy management interface complete with RESTful API to enable automated third-party application integration.

About

Turbine is an easy to set-up and manage load-balancer and reverse proxy management interface complete with a RESTful API to enable automated third-party application integration.

Turbine provides a super-simple RESTful API to enable easier integration with third-party web hosting control panels such as Sentora and other software.

Installation

You can install the latest 'development' snapshot by running the following on your Linux server (currently Ubunu 12.04 LTS is the only supported OS) but other versions may work too!

wget https://github.com/bobsta63/turbine/archive/master.zip

unzip master.zip

cp turbine-master/scripts/install.sh ./install.sh

chmod +x install.sh

sudo ./install.sh

When asked what directory to install from, type:

./turbine-master

That will then install all of the required packages etc. once completed you should be able to login with the default username and password (being: admin/password) and then add rules as required.

Features

Easy to install simply use the commands above to provision and install your reverse proxy/load balancer.

RESTful integration utilise third-party software to manage your sites/configurations or implement functionality into your own systems/software.

System requirements

  • Ubuntu Server 14.04 LTS (or 12.04 LTS)

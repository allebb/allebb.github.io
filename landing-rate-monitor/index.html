LRM (Landing Rate Monitor)

LRM enables you to monitor and track your aircraft landing rates within your flight simulator. The software is designed to filter out landing bounces and will track your initial landing rate.

Screenshots

Features

Easy to install use the installer to install it on your machine - you'll be up and running in a few minutes!

Lightweight only a few megabytes of space required on your machine!

Easy to use just run the application and it'll track your landing rate.

Run it on a remote machine LRM can be run on a remote machine when using WideFS.

System requirements

  • Microsoft FSX or Lockheed Martin Prepar3d
  • FSUIPC client
  • 10MB free disk space

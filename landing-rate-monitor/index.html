LRM (Landing Rate Monitor)

LRM is a free tool that enables you to monitor and track your flights and aircraft landing rates within your flight simulator. The software is designed to filter out landing bounces and will track your initial landing rate.

Screenshots

Features

  • Easy to install use the installer to install it on your machine - you'll be up and running in a few minutes!
  • Lightweight only a few megabytes of space required on your machine!
  • Easy to use just run the application and it'll track your take-off and landing rate.
  • Run it on a remote machine LRM can be run on a remote machine when using WideFS.
  • Display the FPM landing and take-off rate in-sim No more having to minimise your sim to see the landing rate, you can choose to show the landing rate (FPM) directly in the sim!
  • Flight logbook linking the LRM client to a free LRMLive.com account enables you to record all your flights, routes flown and landing statistics.
  • Web-based GPS tracking you can monitor your aircraft position on any device, anywhere...

System requirements

  • Microsoft FSX, FSX Steam Edition or Lockheed Martin Prepar3d
  • FSUIPC client
  • 8MB free disk space

Change log

v3.0 - Released 10th October 2016

  • Additional information now shown for landings (speed, pitch, bank and wind conditions)
  • Optional "BlackBox" mode, that when enabled, it will start LRM at login and will monitor in the background, no need to start it manually anymore!
  • Personal pilot flight records (when linked to an LRMLive.com account).
  • Web-based GPS tracking (when linked to an LRMLive.com account)
  • Display take-off statistics in-sim.
  • Ability to customise how long the in-sim messages are displayed for.

v2.4 - Released 10th August 2016

  • Fixes an issue affecting French regional settings on Microsoft Windows® (issues #2 and #3).
  • Adds new option to show extra landing statistics in the sim on landing (feature request #4).
  • General tweaks and improvements.

v2.3 - Released 6th March 2016

  • Adds new option to auto-close LRM when the Sim is closed.
  • Adds new option to check for LRM updates on start-up.

v2.2 - Released 7th February 2016

  • Pitch/bank angles and wind direction/speed now only uploaded to the LRMLive site instead of outputted to the client/screen (less clutter) as requested by users.

v2.1 - Released 31st January 2016

  • Now captures and displays pitch/bank angles and wind direction/speed
  • Updates API and Map URLs to use the new domain address.

v2.0 - Released 31st March 2015

  • Fixed the update checker tool to correctly report if a new version is available.
  • Added option to display the landing rate (FPM) in the simulator (green banner)
  • Added option to send landing rate and aircraft position to the LRM live map.
  • Various performance improvements.

  • Initial release.

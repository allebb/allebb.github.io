A320 Fuel Calculator

A simple calculation tool designed for flight simulation, this tool calculates the required fuel amount for the Airbus A320 (although should work with A318, A319 and A321 also).

Screenshots

Features

  • Calculations based on the real documentation for the Airbus A319/A320 as well as the EASA OPS 1.255 fuel policy.
  • Contingency fuel (CONT) automatically generated (based on EASA OPS)
  • Fuel weight and percentages displayed and distrubuted over the fuel tanks as recommended by Airbus Ops (outside to centre).

System requirements

  • .NET Framework 4.5
  • 10MB free disk space
